Apa Itu HTML?

Pertama-tama kita harus tahu apa itu HTML, HTML adalah sebuah bahasa markup yang biasanya digunakan untuk membuat sebuah halaman web. Nah, siapa sih penciptanya? Penciptanya adalah Tim Barners Lee pada tahun 1991. Untuk saat ini versi terbarunya yaitu versi HTML5 yang dirilis tahun 2014. HTML adalah kependekan dari Hypertext Markup Language. Nah dari kata Markup berarti HTML bukanlah bahasa pemrograman. Jadi, clear ya yang bilang bahwa HTML itu bahasa pemrograman berarti mereka salah ya brok. Kalau kita analogikan dalam tubuh manusia, HTML bisa kita analogikan sebagai kerangka atau tulang dari manusia. Kenapa sih kita pakai HTML?

Kelebihan HTML

Kelebihan yang kita dapatkan saat menggunakan HTML antara lain:

  • Bersifat Open Source sehingga gratis untuk digunakan
  • Mudah disambungkan dengan bahasa lain contohnya JavaScript dan PHP
  • Bisa dijalankan oleh hampir semua browser
  • Mudah dipelajari oleh web developer pemula

Nah diatas itu adalah kelebihan dari HTML, tetapi bukan berarti tidak ada kekurangan nya ya brok.

Kekurangan HTML

Sebelumnya kita sudah membahas kelebihannya nih, nah sekarang kita bahas kekurangannya ya. Apa aja sih kekurangannya:

  • Hanya diimplementasikan pada web statis
  • Tidak bisa menjalankan logika seperti pertambahan, pengurangan, dan sebagainya
  • Tidak bisa terhubung dengan database

Nah itulah kelebihan dan kekurangan yang dimiliki oleh HTML. Selanjanjutnya apa sih yang ada di dalam HTML itu?

Isi dari HTML

Isi yang ada di HTML disebut dengan tag seperti contoh <p></p>. Ada beberapa tag yang dimiliki oleh HTML, contohnya:

Header

Header adalah tag yang digunakan untuk menampilkan judul atau kepala dari sebuah halaman. Tag header biasanya ditulis dengan <header></header>. Isi dari tag header biasanya ada tag <title> untuk menampilkan judul dari halaman

Body

Selanjutnya ada tag body yang dinyatakan dengan <body></body>. Tag body merupakan tubuh atau isi dari dokumen HTML. Nah di tag body tuh kita bisa berkreasi kaya pake tag <h1></h1> yang digunakan untuk membuat heading dari halaman web atau tag <p></p> untuk membuat paragraf di halaman web. lalu ada tag <img> untuk menyisipkan gambar ke halaman web, dan masih banyak lagi.

Atribut

Yang Terakhir adalah atribut dimana tag awal bisa memiliki beberapa atribut untuk menyatakan karakteristik dari tag itu sendiri. Contohnya, <p style="text-align: left></p> digunakan untuk membuat teks rata kiri di satu paragraf. Tag nya adalah <p> dan atributnya ada "style" dengan nilai "text-align: left".
