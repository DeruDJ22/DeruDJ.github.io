Apa Itu HTML?

Pertama-tama kita harus tahu apa itu HTML, HTML adalah sebuah bahasa markup yang biasanya digunakan untuk membuat sebuah halaman web. Nah, siapa sih penciptanya? Penciptanya adalah Tim Barners Lee pada tahun 1991. Untuk saat ini versi terbarunya yaitu versi HTML5 yang dirilis tahun 2014. HTML adalah kependekan dari Hypertext Markup Language. Nah dari kata Markup berarti HTML bukanlah bahasa pemrograman. Jadi, clear ya yang bilang bahwa HTML itu bahasa pemrograman berarti mereka salah ya brok. Untuk lebih mudah ibaratkan saja HTML itu adalah kerangka atau tulang dari manusia. Kenapa sih kita pakai HTML?

Kelebihan HTML

Kelebihan yang kita dapatkan saat menggunakan HTML antara lain:

  • Bersifat Open Source sehingga gratis untuk digunakan
  • Mudah disambungkan dengan bahasa lain contohnya JavaScript dan PHP
  • Bisa dijalankan oleh hampir semua browser
  • Mudah dipelajari oleh web developer pemula

Nah diatas itu adalah kelebihan dari HTML, tetapi bukan berarti tidak ada kekurangan nya ya brok.

Kekurangan HTML

Sebelumnya kita sudah membahas kelebihannya nih, nah sekarang kita bahas kekurangannya ya. Apa aja sih kekurangannya:

  • Hanya diimplementasikan pada web statis
  • Tidak bisa menjalankan logika seperti pertambahan, pengurangan, dan sebagainya
  • Tidak bisa terhubung dengan database

Nah itulah kelebihan dan kekurangan yang dimiliki oleh HTML. Selanjanjutnya apa sih yang ada di dalam HTML itu?

Isi dari HTML

Isi yang ada di HTML disebut dengan tag seperti contoh <p></p>. Ada beberapa tag yang dimiliki oleh HTML, contohnya:

Header

Header adalah tag yang digunakan untuk menampilkan judul atau kepala dari sebuah halaman. Tag header biasanya ditulis dengan <header></header>. Isi dari tag header biasanya ada tag <title> untuk menampilkan judul dari halaman

Body

Selanjutnya ada tag body yang dinyatakan dengan <body></body>. Tag body merupakan tubuh atau isi dari dokumen HTML. Nah di tag body tuh kita bisa berkreasi kaya pake tag <h1></h1> yang digunakan untuk membuat heading dari halaman web atau tag <p></p> untuk membuat paragraf di halaman web. lalu ada tag <img> untuk menyisipkan gambar ke halaman web, dan masih banyak lagi.

Atribut

Yang Terakhir adalah atribut dimana tag awal bisa memiliki beberapa atribut untuk menyatakan karakteristik dari tag itu sendiri. Contohnya, <p style="text-align: left></p> digunakan untuk membuat teks rata kiri di satu paragraf. Tag nya adalah <p> dan atributnya ada "style" dengan nilai "text-align: left".

Apa itu CSS?

CSS atau Cascading Style Sheets adalah bahasa pemrograman yang digunakan untuk membuat tamplilan, mengatur tata letak, warna dan font dari website kita. CSS dibuat dan dikembangkan oleh W3C atau World Wide Web Consortium pada tahun 1996 dan versi terbarunya adalah CSS 2.1 yang diluncurkan pada tahun 2016. Kenapa sih kita harus pake CSS brok?

Kegunaan CSS

CSS digunakan supaya tampilan website kita jadi lebih menarik tidak statis saja. Misalnya kalau hanya pake HTML tuh cuman tulisan doang nah dengan menggunakan CSS kita bisa mengubah warna background-nya, lalu bisa mengubah font dari paragraf yang kita tulis di HTML, lalu bisa mengubah warna dari font HTML, dan lain sebagainya. Jadi kalau kita analogikan biar lebih mudah, CSS itu bisa dibilang kulit manusia. Lalu gimana sih cara pake CSS?

Cara Menggunakan CSS

Cara menggunakan CSS ada 3 macam, yaitu dengan cara CSS inline, CSS Eksternal, dan CSS Internal. Gimana sih cara penulisannya brok?

CSS Inline

Yang pertama ada CSS inline.

Apa sih CSS Inline itu brok? CSS Inline adalah kode CSS yang ditempatkan di dalam tag HTML contohnya <h1 style="color:blue;text-align:center;">Heading 1</h1> dari kode itu bisa kita dapatkan bahwa tag h1 itu ada sebuah atribut yang namanya style untuk mengubah warna dan membuat textnya berada di tengah. Tapi ada kelebihan dan kekurangan ga sih brok kalo pake CSS Inline? Tentu ada dong brok berikut kelebihan dan kekurangannya nih brok.

Kelebihan
  • Gampang dan cepet buat diterapin ke satu elemen doang
  • Mantep buat testing atau debugging style tertentu
  • Gak perlu ribet nyari-nyari file CSS lain buat ngubah style
Kekurangan
  • Ribet banget kalo websitenya gede brok, karena setiap tag harus ada atribut style
  • Gak efisien, soalnya lu mesti nulis style yang sama berulang-ulang
  • HTML jadi kecampur sama style, berantakan jadinya

CSS Eksternal

Selanjutnya ada CSS Eksternal. Apa lagi itu brok CSS Eksternal? CSS Eksternal adalah CSS yang dibuat diluar kode HTML dan kita memanggilnya ke dalam kode HTML supaya HTML kita bisa nyambung ke CSS kita. Cara menggunakannya di tag header kita menggunakan tag style untuk memanggil CSS. Contoh 
<header>
<link rel="stylesheet" href="style.css">
</header>

Dari kode diatas "link" adalah sebuah tag untuk memanggil file dari CSS nya brok, lalu style.css adalah nama filenya nah didalam file tersebut kita bisa mendekorasi halaman HMTL kita seperti yang kita inginkan contoh penulisan di dalam file CSS nya gini brok 

body {
  background-color: #f2f2f2;
}
h1 {
  color: red;
  margin-left: 20px
}

Nah kan sudah kita bahas cara menggunakannya tapi ada kelebihan dan kekurangannya ga sih brok? Tentu saja ada brok, berikut kelebihan dan kekurangan dari CSS Eksternal.

Kelebihan
  • Rapi dan terorganisir. Semua style ditaruh di satu file, jadi gak berantakan brok
  • Mudah dikelola. Kalo mau ganti style, tinggal edit satu file doang. Gak ribet
  • Reuse style. Bisa dipake di banyak halaman sekaligus. Hemat waktu, brok
  • Loading lebih cepat. Browser bisa cache file CSS-nya, jadi halaman-halaman berikutnya loading lebih cepat
Kekurangan
  • Butuh waktu loading awal. Kalo file CSS-nya besar, bisa bikin loading awal jadi lebih lambat
  • Gak bisa testing instan. Kalo mau lihat perubahan style, harus refresh halaman
  • Bergantung pada file. Kalo file CSS-nya hilang atau error, semua style bisa berantakan
  • Keterbatasan akses. Kalo ada yang mau edit style, harus punya akses ke file CSS-nya

CSS Internal

Yang terakhir adalah CSS Internal atau sering disebut Embedded nih brok. Apaan tuh CSS Internal brok? CSS Internal tuh bisa dibilang sama seperti CSS Inline brok nah tapi dia tidak di tulis di baris yang sama, namun dia ditulis di tag head dan cara penulisannya menggunakan tag style. Contohnya 
<head>
<style>
body {
  background-color: #f2f2f2;
}
h1 {
  color: red;
  margin-left: 20px
}
</style>
</head>
Kelebihan
  • Gampang diimplementasi. Cukup taruh di dalam tag <style> di bagian <head> HTML, beres
  • Cocok buat proyek kecil. Kalo website-nya gak terlalu besar, CSS Internal bisa jadi pilihan yang oke brok
  • Gak perlu file terpisah. Semua style ada di satu tempat, jadi gampang dicari
  • Bisa override style dari CSS eksternal. Kalo mau ganti style tertentu, tinggal tulis di sini
Kekurangan
  • Gak efisien untuk website besar. Kalo banyak halaman, harus nulis style di setiap halaman, ribet
  • Bisa bikin HTML jadi berantakan. Kode CSS campur sama HTML, jadi kurang rapi
  • Keterbatasan performa. Kalo banyak style, bisa bikin loading halaman jadi lebih lambat

Apa Itu JavaScript (JS)?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Dengan JS, Anda dapat menangani event, memanipulasi DOM, dan bahkan mengakses API eksternal untuk memperkaya pengalaman pengguna.
